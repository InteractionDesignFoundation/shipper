# Shipper

> IxDF Release helper based on GitHub API

## Project setup
```sh
yarn install
```

### Compiles and hot-reloads for development
```sh
yarn serve
```

### Compiles and minifies for production
```sh
yarn build
```

### Lints and fixes files
```sh
yarn lint
```

### Run tests
```sh
yarn test
```
